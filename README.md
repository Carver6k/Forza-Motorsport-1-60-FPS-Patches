## Installation

If you're applying the patch to the base game, use forza_60fps_1.05.03.28.59858.ips.
If you're applying the patch to the title update, use forza_60fps_1.05.10.17.60164.ips.

This does not support the XBLA version yet, but you can search for the bytes "02 00 00 00 02 00 00 80" and replace the first 02 with 01.
